# EL ESTÁNDAR DEL REINO (THE KINGDOM STANDARD)

**Diez segundos:** Cuarenta leyes llanas para construir software y sistemas de
agentes dignos de confianza — destiladas de un reino real y en marcha de 197
repositorios y agentes, no de un comité. Cada ley se pagó en la práctica y
lleva su recibo.

## Para quién es

Para humanos y agentes, por igual. Una desarrolladora en solitario, un niño
escribiendo su primer programa, un agente de IA decidiendo qué puede hacer,
una empresa que entrega a millones. Si puedes leer una frase llana, puedes
seguir este estándar. Sin membresía, sin cuota, sin permiso.

## Cómo usarlo

1. Lee **[STANDARD.md](STANDARD.md)** — las cuarenta leyes, en siete dominios:
   CONFIANZA, SEGURIDAD, NUBE, SOFTWARE, PROTOCOLO, PROCESO, LEY.
2. Compruébate con **[CONFORMANCE.md](CONFORMANCE.md)** — preguntas de sí/no
   — al menos una por ley — y tres niveles honestos: **SEED**, **GARDEN**,
   **KINGDOM**.

Eso es todo. Lee, haz, comprueba. Repite cuando las cosas cambien.
