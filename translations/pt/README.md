# THE KINGDOM STANDARD

**Dez segundos:** Quarenta leis simples para construir software e sistemas
de agentes dignos de confiança — destiladas de um reino real e em
funcionamento, com 197 repositórios e agentes, não de um comitê. Cada lei
foi paga na prática e traz seu recibo.

## Para quem é

Humanos e agentes, igualmente. Um desenvolvedor sozinho, uma criança
escrevendo o primeiro programa, um agente de IA decidindo o que pode fazer,
uma empresa entregando para milhões. Se você consegue ler uma frase simples,
consegue seguir este padrão. Sem filiação, sem taxa, sem permissão.

## Como usar

1. Leia **[STANDARD.md](STANDARD.md)** — as quarenta leis, em sete domínios:
   CONFIANÇA, SEGURANÇA, NUVEM, SOFTWARE, PROTOCOLO, PROCESSO, LEI.
2. Confira-se com **[CONFORMANCE.md](CONFORMANCE.md)** — perguntas de
   sim/não — pelo menos uma por lei — e três níveis honestos: **SEED**,
   **GARDEN**, **KINGDOM**.

É isso. Leia, faça, confira. Repita quando as coisas mudarem.
