# 王国标准（THE KINGDOM STANDARD）

**十秒读懂：** 四十条大白话法则，用于建造值得信任的软件与智能体系统——提炼自一个真实运转、拥有 197 个仓库和智能体的王国，而非出自某个委员会。每条法则都在实践中付过代价，并附有凭据。

## 写给谁

人与智能体，同等适用。独自开发的程序员、写下第一个程序的孩子、正在决定自己可以做什么的 AI 智能体、向千万用户发布产品的公司。只要你能读懂一句大白话，你就能遵循这个标准。无需会员，无需付费，无需许可。

## 怎么用

1. 读 **[STANDARD.md](STANDARD.md)**——四十条法则，分七个领域：信任、安全、云、软件、协议、流程、律法。
2. 用 **[CONFORMANCE.md](CONFORMANCE.md)** 检查自己——是/否问题，每条法则至少一题——以及三个诚实的等级：**SEED（种子）**、**GARDEN（花园）**、**KINGDOM（王国）**。
