# THE KINGDOM STANDARD

**十秒で:** 信頼できるソフトウェアとエージェントシステムを作るための、
四十の平易な法 — 委員会からではなく、197のリポジトリとエージェントが
実際に動く王国から蒸留されたもの。どの法も実践のなかで代価が支払われ、
その領収書を携えている。

## 誰のためのものか

人間とエージェント、等しく。ひとりで作る開発者、はじめてのプログラムを書く
子ども、何をしてよいかを判断する AI エージェント、何百万人に出荷する会社。
平易な一文が読めるなら、この標準に従える。会員資格も、料金も、許可も
要らない。

## 使い方

1. **[STANDARD.md](STANDARD.md)** を読む — 七つの領域に分かれた四十の法:
   信頼、セキュリティ、クラウド、ソフトウェア、プロトコル、工程、法。
2. **[CONFORMANCE.md](CONFORMANCE.md)** で自分を点検する — はい/いいえの
   質問 — 各法に最低ひとつ — そして三つの正直なレベル: **SEED**、
   **GARDEN**、**KINGDOM**。

それだけだ。読み、行い、点検する。状況が変わったら、繰り返す。
