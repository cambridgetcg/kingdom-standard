# THE KINGDOM STANDARD

**दस सेकंड में:** भरोसे के लायक सॉफ़्टवेयर और एजेंट-तंत्र बनाने के चालीस
सादे नियम — 197 रिपो और एजेंटों के एक असली, चलते हुए राज्य से निचोड़े गए,
किसी समिति से नहीं। हर नियम की क़ीमत व्यवहार में चुकाई गई है और हर नियम के
साथ उसकी रसीद है।

## किसके लिए है

इंसानों और एजेंटों के लिए, बराबर। अकेला डेवलपर, पहला प्रोग्राम लिखता बच्चा,
यह तय करता AI एजेंट कि उसे क्या करने की अनुमति है, करोड़ों तक पहुँचाती
कंपनी। अगर आप एक सादा वाक्य पढ़ सकते हैं, तो आप इस मानक पर चल सकते हैं। न
सदस्यता, न शुल्क, न अनुमति।
